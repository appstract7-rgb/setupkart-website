Fecha: febrero 2026

2. Descripción de servicios

SetupKart Pro es una aplicación móvil para el registro, gestión y análisis de datos de configuración de kart, incluyendo:

  • Registro de ajustes de neumáticos, chasis, motor y carburador
  • Datos meteorológicos y de circuito
  • Registro y análisis de tiempos por vuelta
  • Exportación e importación de datos (.kart)
  • Entrada de voz para notas

La App está disponible en alemán, inglés, francés, italiano, español y neerlandés.

Política de privacidad

Estado: febrero 2026

1. Introducción

La protección de sus datos personales es importante para nosotros. Esta política de privacidad le informa sobre qué datos se recopilan al utilizar la app «SetupKart Pro» y cómo se utilizan.

Responsable del tratamiento de datos: Hermann Kopp, Suiza (email: appstract7@gmail.com).

2. Principio: Almacenamiento local de datos

SetupKart Pro fue desarrollado según el principio «Privacy by Design». Todos los datos de setup se almacenan localmente en su dispositivo por defecto.

  • Ningún dato personal transmitido a servidores externos (excepto consulta meteorológica opcional, ver 3.6)
  • Ninguna cuenta de usuario requerida
  • Ninguna herramienta de seguimiento utilizada
  • Ningún servicio de análisis o publicidad integrado

Si el usuario activa futuras funciones cloud opcionales, los datos podrán almacenarse en servidores externos. El usuario será informado previamente.

3. ¿Qué datos se procesan?

3.1 Datos de setup (locales)
Todos los datos técnicos introducidos en la app se almacenan exclusivamente en su dispositivo.

3.2 Ubicación GPS (opcional)
La app puede utilizar su ubicación GPS para identificar automáticamente el circuito. Estos datos se procesan solo localmente (excepto para la consulta meteorológica opcional, ver 3.6).

3.3 Entrada de voz (opcional)
La función de entrada de voz utiliza el reconocimiento de voz de su dispositivo, según las condiciones del fabricante (Google/Android).

3.4 ID de dispositivo (licencia)
Se utiliza un ID de dispositivo para vincular el código de activación a su dispositivo, exclusivamente para la gestión de licencias.

3.5 Datos de contacto (compra)
Su dirección de email se utiliza para la entrega del código de activación y no se comparte con terceros.

3.6 Datos meteorológicos (opcional)
La app puede, a petición del usuario, obtener datos meteorológicos actuales a través de la API Open-Meteo. Las coordenadas GPS (latitud, longitud) se transmiten al servicio open-meteo.com. No se transmiten datos personales. Open-Meteo es un servicio meteorológico de código abierto que no requiere registro. Más información: open-meteo.com.

4. Exportación y uso compartido de setup

La exportación y el uso compartido de datos (.kart) los inicia el usuario. El proveedor no tiene influencia sobre los destinatarios.

5. Google Play Store

Si obtiene la app a través del Google Play Store, también se aplica la política de privacidad de Google.

6. Derechos de los usuarios

Tiene derecho de acceso, rectificación, supresión, limitación del tratamiento y portabilidad de datos. Contacto: appstract7@gmail.com.

7. Seguridad de los datos

El almacenamiento local ofrece un alto nivel de seguridad ya que no se realiza ninguna transmisión por Internet (con excepción de la consulta meteorológica opcional).

8. Período de conservación

Los datos locales permanecen en su dispositivo hasta que los elimine o desinstale la app.
