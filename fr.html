Politique de confidentialité

État : février 2026

1. Introduction

La protection de vos données personnelles nous tient à cœur. Cette politique de confidentialité vous informe des données collectées lors de l'utilisation de l'app «SetupKart Pro» et de leur utilisation.

Responsable du traitement des données : Hermann Kopp, Suisse (e-mail : appstract7@gmail.com).

2. Principe : Stockage local des données

SetupKart Pro a été développé selon le principe «Privacy by Design». Toutes vos données de setup sont stockées localement sur votre appareil par défaut.

  • Aucune donnée personnelle transmise à des serveurs externes (sauf requête météo optionnelle, voir 3.6)
  • Aucun compte utilisateur requis
  • Aucun outil de suivi utilisé
  • Aucun service d'analyse ou de publicité intégré

Si l'utilisateur active de futures fonctions cloud optionnelles, les données pourront être stockées sur des serveurs externes. L'utilisateur en sera informé au préalable.

3. Quelles données sont traitées ?

3.1 Données de setup (locales)
Toutes les données techniques saisies dans l'app sont stockées exclusivement sur votre appareil.

3.2 Position GPS (optionnel)
L'app peut utiliser votre position GPS pour identifier automatiquement le circuit. Ces données sont traitées localement uniquement (sauf pour la requête météo optionnelle, voir 3.6).

3.3 Saisie vocale (optionnel)
La fonction de saisie vocale utilise la reconnaissance vocale de votre appareil, selon les conditions de votre fabricant (Google/Android).

3.4 ID d'appareil (licence)
Un ID d'appareil est utilisé pour lier le code d'activation à votre appareil, exclusivement pour la gestion des licences.

3.5 Données de contact (achat)
Votre adresse e-mail est utilisée pour la livraison du code d'activation et n'est pas transmise à des tiers.

3.6 Données météo (optionnel)
L'app peut, à la demande de l'utilisateur, récupérer les données météo actuelles via l'API Open-Meteo. Les coordonnées GPS (latitude, longitude) sont transmises au service open-meteo.com. Aucune donnée personnelle n'est transmise. Open-Meteo est un service météo open-source ne nécessitant aucune inscription. Plus d'informations : open-meteo.com.

4. Export et partage de setup

L'exportation et le partage de données (.kart) sont initiés par l'utilisateur. Le fournisseur n'a aucune influence sur les destinataires.

5. Google Play Store

Si vous obtenez l'app via le Google Play Store, la politique de confidentialité de Google s'applique également.

6. Droits des utilisateurs

Vous avez le droit d'accès, de rectification, de suppression, de limitation du traitement et de portabilité des données. Contact : appstract7@gmail.com.

7. Sécurité des données

Le stockage local offre un haut niveau de sécurité car aucune transmission via Internet n'a lieu (à l'exception de la requête météo optionnelle).

8. Durée de conservation

Les données locales restent sur votre appareil jusqu'à leur suppression ou la désinstallation de l'app.
